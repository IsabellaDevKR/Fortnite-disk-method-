# Requirements
- 10 IQ ( atleast )
- SSD / HDD (Fast enough to run fortnite)
- Atleast 100GB of storage on that SSD / HDD ( enough to install fn )

  # STEP 1.
  - Open Disk Management
 
  # STEP 2.
  - Press on your ssd / HDD with the free storage once

  # STEP 3.
  - Go to the very top, go to "Action", press create VHD, set the location to your desktop and name it "1", now change the size to 100GB, press ok, a bar should appear wait it out.
 
  # STEP 4.
  - Repeat step 3, make another one (Name it "2" this time)
 
  # STEP 5.
  - Once done you should have 2 unallocated "drives", right click one of them like shown in the image
  <img width="516" height="270" alt="image" src="https://github.com/user-attachments/assets/88d38721-c136-4081-8ea7-f6d06923bf10" />
  
  Press "Initialize", select MBR (NOT GPT), make sure you selected both drives. Press OK, now exit disk management

  # STEP 6.
  - Now press your windows key, type "Manage Storage Spaces", now press create, select both drives and press "Create Pool", now you will see alot of things, but calm down, dont panic
 
  # STEP 7.
  - Create the name at the top, and the desired letter, I always choose W.. cus.. I am a W, anyways the most important thing is the size, so in the size if u created both 100s, then you should type 200, both sizes added, whatever the site you did for your game.. in my case 100+100 = 200, press create.
 
  # STEP 8.
  - 90% completed, now check your serials!

  # VERY IMPORTANT NOTES
  - Everytime you start your pc you must start the VHD drives (1 and 2) created on your desktop, just double click them an error will appear, just press OK, once you start both. The drives will start and then epic games will read the game from that disk since its installed on it... Avoid opening epic games before launching the drives since it wont find the game and will force you to reinstall your game.
